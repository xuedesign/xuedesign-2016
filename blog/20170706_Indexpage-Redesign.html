淺談網站首頁設計

網站改造 版面設計 Landing Page

Elmo Woo
Jul 06, 2017

網站首頁真的很重要嗎?

網站首頁代表一個網站的門面，有八成透過搜尋引擎進來的民眾，第一個抵達頁面就是首頁。很自然的首頁就擔負起迎賓的工作，在首頁中你必須要清楚的傳達公司營業項目、形象理念及達到引導至其他內頁的作用，在過去許多公司喜歡使用 Flash 來製作精美的動畫，認為動畫可以傳達給客戶耳目一新的公司形象，但殊不知許多訪客沒有耐心看完你那落落長的過場動畫，而選擇按下「Skip」進入內頁，這樣反而造成擾民的負面印象。加上近年 Flash 技術已逐漸被淘汰，許多手機或平板裝置已宣告不再支援 Falsh 的運作，因此屬於 Flash 的黃金時代已經過去。

現代網頁技術日新月異，幾乎每天都有新的技術在發表，首頁轉場動畫早已被 CSS 及 Javascript 所取代，兩個組合起來所能達到的效果，可不比 Flash 差勁，也對於搜尋引擎的爬搜較為友善，有助於網站 SEO (網站自然排序)優化的提升，所以現代較多採用這樣的規格來執行。

首頁的版面設計

這邊我以一個過往經手的案例來做說明，風林國際是一家行銷海外的塑膠料件供應商，為提供國外訪客感受到公司專業的第一印象，與我們討論過後，希望在首頁設計的部分，以大篇幅的形象圖及宣傳口號為主，加上導覽列與聯絡資訊為輔。因此產生了下面的設計，以外國商務人士的手勢姿態導引，使訪客隨著動線首先看到公司 LOGO，接著是宣傳 Slogan，最後是公司聯絡資料及其他單元頁面的連結，從上到下一氣呵成。

發現問題點

上線一段時間後，發現首頁在當初規劃上有些隱藏的問題未發現，導致資訊傳遞的不夠完善。比如說公司營業項目不明確、可以增加公司技術優勢說明，來加深訪客對於公司的專業印象等對外資訊的宣導，使首頁宣傳效果更為提升。於是乎開始研究，要如何將上述的問題，運用設計融入版面編排之中，我們想到了一般電商很常運用的 Landing Page (商品登陸頁)的模式，似乎可以解決這次的問題。

精準地有效傳達資訊，提高自然轉換率(引起興趣或購買)，這就是 Landing Page 的精髓。

導入Landing Page設計概念

這次導入 Landing Page 的核心概念就在於要增進訪客對於公司的信心，消除他們的疑慮，並喚起後續聯絡公司諮詢產品資訊的意願，所以在版面的配置上，上面先以公司三大主線產品的輪播圖，讓訪客了解公司業務屬性與範圍，接著是公司簡介與技術優勢說明，最後是公司聯絡資訊及內容頁面連結，引導訪客繼續深入閱讀公司其他資訊或聯絡業務人員諮詢產品細節。最後的首頁設計則如下圖所示，並搭配適時的進場效果，讓版面呈現更為生動。
風林國際首頁效果預覽

首頁設計真的很重要!

經過這次的案例輔助說明，希望可以讓大家了解到相較於其他頁面的權重，一個網站的首頁其實肩負著重責大任不可輕視之，若是首頁做得好，業績自然不會少，但也不是任何首頁都套入 Landing Page 概念都適用，畢竟還是要以實際需求來評估網站首頁最適合的呈現方式。當然，也歡迎找我們做網站事前規劃的諮詢，我們會很樂意提供專業意見的。(馬上連絡我們!!)
